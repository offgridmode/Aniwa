# Aniwa

> Universal dataset profiling and intelligence.

Aniwa helps developers, analysts, data engineers, and modern data teams understand datasets quickly through intelligent profiling, quality analysis, statistical summaries, and beautiful reports.

---

## Quick Installation

```bash
pip install aniwa
```

---

## Quick Start

```bash
aniwa profile customers.csv
```

Generate HTML report:

```bash
aniwa profile customers.csv --report html
```
